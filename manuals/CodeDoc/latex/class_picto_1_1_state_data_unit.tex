\subsection{Detailed Description}
Stores data about a particular \hyperlink{class_picto_1_1_transition}{Transition} traversal that occured during a \hyperlink{class_picto_1_1_task}{Task} run for transmission over the network. 

Objects of this class store the Asset\-Id of a \hyperlink{class_picto_1_1_transition}{Transition} that was traversed along with the Data\-Id of the Frame that was presented following that traversal. This information can later be used to reproduce the precise path through the \hyperlink{class_picto_1_1_state_machine}{State\-Machine} that was taken during a given session. \begin{DoxyAuthor}{Author}
