\subsection{Detailed Description}
A simple parameter for holding a boolean value. 

The Value \hyperlink{class_picto_1_1_property}{Property} is runtime editable and appears as a checkbox widget in the \hyperlink{class_property_frame}{Property\-Frame}. The boolean value is accessible through the javascript \char`\"{}value\char`\"{} property as\-: 
\begin{DoxyCode}
BooleanParameterName.value = \textcolor{keyword}{false};
var boolVal = BooleanParameterName.value;
\end{DoxyCode}
