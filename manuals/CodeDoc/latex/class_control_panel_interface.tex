\subsection{Detailed Description}
The interface between the \hyperlink{namespace_picto}{Picto} \hyperlink{class_director}{Director} and a Hardware Control panel. 

A hardware control panel controls the \hyperlink{class_director}{Director} name, reward/flush durations, and offers the capability to start a reward or start/stop a flush. This class provides interfaces for all of these capabilities and should be inherited to implement that interface for particular types of hardware. The \hyperlink{namespace_picto}{Picto} Engine uses this class to inform the front panel when these \hyperlink{class_director}{Director} properties have changed, and to change these \hyperlink{class_director}{Director} properties when asked by the control panel to do so. \begin{DoxyAuthor}{Author}
Joey Schnurr, Mark Hammond, Matt Gay
