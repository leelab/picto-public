Implements \hyperlink{class_picto_1_1_property_a97d52011d6db190c5c28a21d76ac1d3b}{Property\-::string\-To\-Variant()} to convert the input string to a variant color. 

The input string may be in the format returned from \hyperlink{class_picto_1_1_color_property_a70dc856281122de27ff0ab73a1cc3940}{variant\-To\-String()} or in the \#rrggbb where rr, gg and bb are hex values from 0 to F\-F. This is supported since Q\-Variant color -\/$>$ Q\-String conversion returns colors in that format.
