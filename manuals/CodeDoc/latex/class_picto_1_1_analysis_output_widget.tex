\subsection{Detailed Description}
The base class of widgets that display a representation of the output of an \hyperlink{class_picto_1_1_analysis_output}{Analysis\-Output} object. 

These widgets are displayed in an \hyperlink{class_picto_1_1_analysis_output_widget_container}{Analysis\-Output\-Widget\-Container} for monitoring of \hyperlink{class_picto_1_1_analysis}{Analysis} outputs from within the \hyperlink{namespace_picto}{Picto} application. Different \hyperlink{class_picto_1_1_analysis_output}{Analysis\-Output} classes will be associated with different Analysis\-Widgets, for example, an \hyperlink{class_picto_1_1_analysis_file_output}{Analysis\-File\-Output} will be associated with an \hyperlink{class_picto_1_1_analysis_file_output_widget}{Analysis\-File\-Output\-Widget}, whereas an Analysis\-Raster\-Output might still save its data to file, but be viewable using an Analysis\-Raster\-Plot\-Widget. \begin{DoxyAuthor}{Author}
