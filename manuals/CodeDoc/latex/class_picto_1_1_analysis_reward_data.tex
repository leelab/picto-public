\subsection{Detailed Description}
An \hyperlink{class_picto_1_1_analysis_data_source}{Analysis\-Data\-Source} for gathering reward timing data. 

This object can be used to query reward onset times as well as their durations. \begin{DoxyNote}{Note}
This class uses a \hyperlink{class_picto_1_1_reward_reader}{Reward\-Reader} as its main data source.
