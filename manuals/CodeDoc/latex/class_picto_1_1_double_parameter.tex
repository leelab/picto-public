\subsection{Detailed Description}
A simple parameter for holding a floating point numeric value. 

Min and Max Properties are also provided to constrain the numeric value to a set range. The Value \hyperlink{class_picto_1_1_property}{Property} is runtime editable and appears as a numeric input widget in the \hyperlink{class_property_frame}{Property\-Frame}. The numeric value is accessible through the javascript \char`\"{}value\char`\"{} property as\-: 
\begin{DoxyCode}
DoubleParameterName.value = 0.26;
var numVal = DoubleParameterName.value;
\end{DoxyCode}
