\subsection{Detailed Description}
An \hyperlink{struct_picto_1_1_output_signal}{Output\-Signal} that handles the value of single pins of an entire port. 

The pin's voltage value is set as a bolean. \begin{DoxyNote}{Note}
Using a \hyperlink{class_picto_1_1_binary_data_output}{Binary\-Data\-Output} object and a \hyperlink{class_picto_1_1_digital_output}{Digital\-Output} object with the same port is undefined. We might want to check for this at some point in validate\-Object. More than one \hyperlink{class_picto_1_1_digital_output}{Digital\-Output} can be used with the same port so long as eash one handles a different pin. 
\end{DoxyNote}
