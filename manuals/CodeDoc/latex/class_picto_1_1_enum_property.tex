\subsection{Detailed Description}
A \hyperlink{class_picto_1_1_property}{Property} for storing enums. 

This is useful for storing a list of options that the designer can select. The list of possible enum values is set by adding a \hyperlink{class_picto_1_1_property}{Property} Attribute with the name \char`\"{}enum\-Names\char`\"{} and a Q\-Variant of a Q\-String\-List containing enum names for the value. This is done using \hyperlink{class_picto_1_1_property_ac70060568119a4c00e23c03a0bc81f41}{Property\-::set\-Attribute()}. \begin{DoxyAuthor}{Author}
