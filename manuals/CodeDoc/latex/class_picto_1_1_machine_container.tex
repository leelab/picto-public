\subsection{Detailed Description}
A base class for everything that can contain transitions and other elements. 

This class should be refactored. It appears the way it does mainly for historical reasons, but essentially what it does is manage transitions and objects in the \hyperlink{class_picto_1_1_state_machine}{State\-Machine} that are not State\-Machine\-Elements or Output\-Elements. There should probably be one class that handles Transitions, and one or more that handles Variables, Parameters, etc.

As it stands however, this is a fairly important class that handles a lot of the code necessary for elements that contain \char`\"{}machines\char`\"{} that includes some form of state transfer. Currently these include \hyperlink{class_picto_1_1_state_machine}{State\-Machine} and \hyperlink{class_picto_1_1_state}{State}. \begin{DoxyAuthor}{Author}
