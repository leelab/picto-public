\subsection{Detailed Description}
Triggers a result according to a logic test that runs each frame, thereby ending the execution of its parent \hyperlink{class_picto_1_1_state}{State}. 

All \hyperlink{namespace_picto}{Picto} \hyperlink{class_picto_1_1_state}{State} elements must include a Control Element of some sort. The control element determines when the \hyperlink{class_picto_1_1_state}{State} ends. During the \hyperlink{class_picto_1_1_state}{State} element's rendering loop, the \hyperlink{class_picto_1_1_control_element_acf73c056833f5da319a0ae046b82b3a8}{is\-Done()} function is called every frame until it returns true. Then the \hyperlink{class_picto_1_1_control_element_a24e223e49d560fa2c3d9e6f02d68bafb}{get\-Result()} function is called to determine the result of the \hyperlink{class_picto_1_1_control_element}{Control\-Element}'s execution(e.\-g. \char`\"{}\-Broke fixation\char`\"{}, \char`\"{}\-Selected target\char`\"{}, etc) \begin{DoxyAuthor}{Author}
