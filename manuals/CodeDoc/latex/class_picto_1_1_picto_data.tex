\subsection{Detailed Description}
The root node of the \hyperlink{namespace_picto}{Picto} design tree. 

This \hyperlink{class_picto_1_1_data_store}{Data\-Store} is the root \hyperlink{class_picto_1_1_data_store}{Data\-Store} of the entire \hyperlink{namespace_picto}{Picto} Design. Its children include \hyperlink{class_picto_1_1_experiment}{Experiment} objects, and any number of \hyperlink{class_picto_1_1_analysis}{Analysis} objects and \hyperlink{class_picto_1_1_u_i_data}{U\-I\-Data} objects. Whereas the \hyperlink{class_picto_1_1_design_root}{Design\-Root} is a convenient package for moving designs around, the \hyperlink{class_picto_1_1_picto_data}{Picto\-Data} node is the real root of the design, included in the actual serialization/deserialization procedure.
