\subsection{Detailed Description}
The main window of the proxy server. 

This is the G\-U\-I for the Proxy application. Since the Proxy application is so simple, this class handles the application state as well. There are only 5 possible states\-:
\begin{DoxyItemize}
\item Init\-State\-: Application starts in this state.
\item Wait\-For\-Connect\-: Application is attempting to connect to the \hyperlink{namespace_picto}{Picto} \hyperlink{class_server}{Server}
\item Wait\-For\-Session\-: Application is waiting for a new session command to come in from the \hyperlink{namespace_picto}{Picto} \hyperlink{class_server}{Server}.
\item Wait\-For\-Device\-: Application is in a session and waiting for the Neural Data Acq. device handled by the plugin to start sourcing data.
\item Running\-: Application is running a Session and sending neural data to the \hyperlink{namespace_picto}{Picto} \hyperlink{class_server}{Server}.
\end{DoxyItemize}
