\subsection{Detailed Description}
An empty event code generator for testing purposes. 

Since we aren't always running with access to a neural recording device, we don't always need to generate event codes. The \hyperlink{class_picto_1_1_null_event_code_generator}{Null\-Event\-Code\-Generator} serves this purpose.
