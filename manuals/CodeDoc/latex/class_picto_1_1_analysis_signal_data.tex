\subsection{Detailed Description}
An \hyperlink{class_picto_1_1_analysis_data_source}{Analysis\-Data\-Source} for gathering signal data. 

This object can be used to query signal data for any input signal as well as metadata including the sub components of the signal and its sample period. The signal tracked is set in the Signal\-Name property, where \char`\"{}\-Position\char`\"{} and \char`\"{}\-Diameter\char`\"{} are the standard signal names, but theoretically, there could be any number of signals with different names. They are defined in the \hyperlink{class_director}{Director}'s \hyperlink{class_hardware_setup}{Hardware\-Setup} by using Engine\-::add\-Signal\-Channel().

\begin{DoxyNote}{Note}
This class uses a \hyperlink{class_picto_1_1_signal_reader}{Signal\-Reader} as its main data source. 
\end{DoxyNote}
