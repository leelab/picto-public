\subsection{Detailed Description}
The \hyperlink{class_test_playback_controller}{Test\-Playback\-Controller} handles the \hyperlink{class_test_playback_controller_a169259eb2f0c71d173efdf1b3f00bef3}{play()}, \hyperlink{class_test_playback_controller_a7699c2c9b554951f3d14214c02496b33}{pause()}, \hyperlink{class_test_playback_controller_a6a4a3a56fea5a3639094abc21b5db3ab}{stop()} interface to an Experiment.

Starts / Resume playback of the current Task. 

When no Task is running, this emits \hyperlink{class_test_playback_controller_ae02259c2ad0ea392389640b93326aebd}{running()} then starts running the Task. The function blocks until the Task ends, either on its own, or because stop was pressed. When that happens, the \hyperlink{class_test_playback_controller_a3ff862f1e1ff73faa49fdab265645625}{stopped()} signal is emitted and the function ends.

If this is called when a Task is already running, it causes the task to resume, and the \hyperlink{class_test_playback_controller_ae02259c2ad0ea392389640b93326aebd}{running()} signal is emitted. \begin{DoxyNote}{Note}
Since the Task runs underneath the \hyperlink{class_test_playback_controller_a169259eb2f0c71d173efdf1b3f00bef3}{play()} function, when this function is used to resume playback, it is actually being called from below another call to \hyperlink{class_test_playback_controller_a169259eb2f0c71d173efdf1b3f00bef3}{play()}. This is confusing, and at some point we need to refactor the Experimental run system so that it is played from within a Qt Event loop. 
\end{DoxyNote}

Stops the currently running task. 

As described in \hyperlink{class_test_playback_controller_a169259eb2f0c71d173efdf1b3f00bef3}{play()}, if a Task is running it means that this is getting called somewhere underneath a call to \hyperlink{class_test_playback_controller_a169259eb2f0c71d173efdf1b3f00bef3}{Test\-Playback\-Controller\-::play()}. The reults of this function will be that the Experiment\-::run\-Task() function in \hyperlink{class_test_playback_controller_a169259eb2f0c71d173efdf1b3f00bef3}{play()} will return, and then execution will continue within that function until the \hyperlink{class_test_playback_controller_a3ff862f1e1ff73faa49fdab265645625}{stopped()} signal is emitted.
