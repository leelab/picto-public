\subsection{Detailed Description}
An base class for classes that can provide Run Notes data. 

This class allows Analyses to gather data about the current run, so that it can be used to provide meta-\/data in \hyperlink{class_picto_1_1_analysis}{Analysis} outputs. \begin{DoxyAuthor}{Author}
