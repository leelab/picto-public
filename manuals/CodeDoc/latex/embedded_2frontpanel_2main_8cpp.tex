\subsection{Detailed Description}
This application controls the front panel on Picto\-Box. The front panel on Picto\-Box consists of a 2 line L\-C\-D and a rotary encoder plus button input device (a Phidget\-Encoder, part number 1052). The application handles user input and communicates with the \hyperlink{namespace_picto}{Picto} engine via a simple T\-C\-P/\-I\-P connection.
