\subsection{Detailed Description}
Handles P\-R\-O\-X\-Y\-L\-I\-S\-T commands. 

Picto\-Workstation sends these commands in order to get a list of all the \hyperlink{namespace_picto}{Picto} Proxy instances that are currently connected to the server. \begin{DoxyAuthor}{Author}
