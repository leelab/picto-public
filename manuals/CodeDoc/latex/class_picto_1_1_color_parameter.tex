\subsection{Detailed Description}
A simple parameter for holding a color value. 

The Value \hyperlink{class_picto_1_1_property}{Property} is runtime editable and appears as a color selection widget with 3 editable values and a color wheel pop-\/up in the \hyperlink{class_property_frame}{Property\-Frame}. The color value is accessible through various javascript properties and functions. For example\-: 
\begin{DoxyCode}
ColorParameterName.value = \textcolor{stringliteral}{"#FF2255"};   \textcolor{comment}{//"#rrggbb"}
fixationPointer.red = ColorParameterName.red;
\end{DoxyCode}
