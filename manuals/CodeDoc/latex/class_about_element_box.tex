\subsection{Detailed Description}
A searchable read-\/only text box that provides information about \hyperlink{namespace_picto}{Picto} Asset functionality. 

This box is used in the picto \hyperlink{class_designer}{Designer} to provide desginers with contextual data including an Overview, Data\-Store properties, Script properties, and Script functions for every selected Asset. \begin{DoxyAuthor}{Author}
