\subsection{Detailed Description}
Handles overall status operations for a \hyperlink{class_proxy_main_window}{Proxy\-Main\-Window} object. 

This class adds to the default \hyperlink{class_component_status_manager}{Component\-Status\-Manager} functionality by implimenting \hyperlink{class_proxy_status_manager_a08b1d3111d19c5fa7fddedf9799577da}{do\-Server\-Update()} to send C\-O\-M\-P\-O\-N\-E\-N\-T\-U\-P\-D\-A\-T\-E commands to the \hyperlink{namespace_picto}{Picto} \hyperlink{class_server}{Server}. It also provides functions for setting/getting the neural data acquisition plugin that will be used to gather data from the Neural D\-A\-Q software for sending to the \hyperlink{class_server}{Server}. \begin{DoxySeeAlso}{See Also}
