\subsection{Detailed Description}
This is a simple object used to store data about the \char`\"{}host Asset\char`\"{} to which an \hyperlink{class_picto_1_1_associate_element}{Associate\-Element} links. 

The object includes two Properties, Parent\-Path and Parent\-Id that store data about the \hyperlink{class_picto_1_1_asset}{Asset} to which the \hyperlink{class_picto_1_1_associate_element}{Associate\-Element} links. Since this object is a required child of an \hyperlink{class_picto_1_1_associate_element}{Associate\-Element}, the \hyperlink{class_picto_1_1_property}{Property} values will be saved with the \hyperlink{class_picto_1_1_associate_element}{Associate\-Element} so that a record will be maintained of the \hyperlink{class_picto_1_1_asset}{Asset} to which the \hyperlink{class_picto_1_1_associate_element}{Associate\-Element} should link. Other functionality is included here too. Once an \hyperlink{class_picto_1_1_asset}{Asset} is linked using the \hyperlink{class_picto_1_1_associate_host_link_ac4a7f2219f146ea54f0c4763ff1eb5e9}{link\-To\-Asset()} function, its signals are connected so that changes in the linked \hyperlink{class_picto_1_1_asset}{Asset} name or id will automatically propogate to this object's \hyperlink{class_picto_1_1_property}{Property} values. update\-Linke\-Path() functionality is also provided for the purpose of \hyperlink{class_picto_1_1_associate_element}{Associate\-Element} export/import. \begin{DoxyAuthor}{Author}
