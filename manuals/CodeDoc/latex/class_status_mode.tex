\subsection{Detailed Description}
A \hyperlink{class_display_mode}{Display\-Mode} that allows the user to monitor the current status of the Front Panel application. 

Displayed information includes Pictobox Name, Server\-Disconnected/\-Server\-Connected/\-Stopped/\-Running status symbol, Reward duration, Flush duration, Default Reward Controller Number, Picto\-Box I\-P address.
