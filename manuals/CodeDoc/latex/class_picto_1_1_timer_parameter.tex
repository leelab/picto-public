\subsection{Detailed Description}
A \hyperlink{class_picto_1_1_timer_parameter}{Timer\-Parameter} for checking how much time has passed between different sections of the \hyperlink{class_picto_1_1_state_machine}{State\-Machine}. 

This Timer works according to Frame resolution. This means that when you call \hyperlink{class_picto_1_1_timer_parameter_ae77dae4dbc2b5309d6edcec4d8ca4471}{restart()} it will set the time of the last frame's first phosphor presentation as if it is time zero. All calls to \hyperlink{class_picto_1_1_timer_parameter_a9a6e24c716f84e5d5b89ccee753d8a5f}{get\-Value()} or uses of the \char`\"{}value\char`\"{} script parameter will return the time of the last frame's first phoshpor presentation with respect to the \hyperlink{class_picto_1_1_timer_parameter_ae77dae4dbc2b5309d6edcec4d8ca4471}{restart()} time. We do things this way because the only real meaningful times in a \hyperlink{namespace_picto}{Picto} experiment are the times at which frames are presented. It is much more logical to return everything in terms of these times rather than in terms of the time at which a certain section of code happened to be executed.

The \hyperlink{class_picto_1_1_timer_parameter}{Timer\-Parameter} is essentially just an interface for a Frame\-Resolution\-Timer. See that class for more detail. \begin{DoxyAuthor}{Author}
