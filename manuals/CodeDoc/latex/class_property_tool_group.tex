Called when all buttons are unpressed and similar situations to disable all button actions that may have been started during a previous \hyperlink{class_property_tool_group_ad83896033e32ef8f2aaf250301c0da55}{do\-Button\-Action()} call. 

For example, as \hyperlink{class_asset_tool_group}{Asset\-Tool\-Group} might have set a new insertion item to be added to the design when the user clicks the canvas. When this function is called, it sets the insertion item back to N\-U\-L\-L.

Returns true if the button at the input button id should be enabled in the widget and false if it should appear grayed out. 

For example, if the number of child assets of a certain type is limited in a particular window, this would return false for that asset's button once the limit is reached.
