\subsection{Detailed Description}
Stores a list of \hyperlink{class_picto_1_1_state_data_unit}{State\-Data\-Unit} objects representing \hyperlink{class_picto_1_1_state_machine}{State\-Machine} Transitions between one frame and the next. 

Objects of this class store the entire \hyperlink{class_picto_1_1_state_machine}{State\-Machine} traversal that occured during the course of a single frame. The traversal is stored as a list of \hyperlink{class_picto_1_1_state_data_unit}{State\-Data\-Unit} objects each of which contains a \hyperlink{class_picto_1_1_transition}{Transition} Asset\-Id. These \hyperlink{class_picto_1_1_state_data_unit}{State\-Data\-Unit} objects are stored with the Data\-Id of the Frame that followed them, and since they each have their own Data\-Id, we can follow the order of \hyperlink{class_picto_1_1_transition}{Transition} traversals accordingly. \begin{DoxyAuthor}{Author}
