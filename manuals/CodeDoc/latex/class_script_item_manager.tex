\subsection{Detailed Description}
Manages creation and layout of Script\-Items for the various Diagram\-Items that use them. 

Diagram\-Items that show Script\-Items present them along their length in the order that the scripts are called with varying dimensions depending on whether we are operating in Analysis or Experiment mode. This dramatically improves design readability, and since there are a number of separate Diagram\-Items that use this paradigm it is useful to create a single class that manages creation and layout of Script\-Items for reuse by all of the Diagram\-Items needing them. This is that class. It shows Entry, Frame and Exit scripts depending on which Asset is being represented. When operating in Analysis mode, Analysis scripts are shown as well. Each script appears in the order that it is called (ie. Analysis\-Entry, Entry, Frame, Analysis\-Frame, Exit, Analysis\-Exit) from top to bottom or left to right. \begin{DoxyAuthor}{Author}
