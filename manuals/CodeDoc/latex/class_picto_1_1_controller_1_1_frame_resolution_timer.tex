\subsection{Detailed Description}
A timer object that returns times in single frame increments. 

\hyperlink{class_picto_1_1_control_element}{Control\-Element} objects and other sections of \hyperlink{namespace_picto}{Picto} require precise timing aligned with frame intervals. This object serves that purpose. The timer always returns times according to the instant at which the latest first phosphor occured. \hyperlink{class_picto_1_1_controller_1_1_frame_resolution_timer}{Frame\-Resolution\-Timer} objects require an underlying \hyperlink{struct_picto_1_1_controller_1_1_frame_tracker}{Frame\-Tracker} object which provides frame timing information. The \hyperlink{class_picto_1_1_controller_1_1_frame_resolution_timer}{Frame\-Resolution\-Timer} is created by a \hyperlink{class_picto_1_1_controller_1_1_frame_timer_factory}{Frame\-Timer\-Factory}, and this factory provides the \hyperlink{struct_picto_1_1_controller_1_1_frame_tracker}{Frame\-Tracker} instance. \begin{DoxyAuthor}{Author}
