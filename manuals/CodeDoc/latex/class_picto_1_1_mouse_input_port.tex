\subsection{Detailed Description}
A signal channel for use with the mouse. 

Although a mouse signal channel is completely useless in an experiment (unless your subject is capable of using a mouse), it is absolutely essential when testing an experiment. When \hyperlink{class_picto_1_1_mouse_input_port_a67e15a3e712644d7a1bd70dcba94d818}{update\-Data\-Buffer()} is called, the channel polls the mouse and grabs the most recent coordinates. This could potentially cause us to miss very quick movements but since \hyperlink{namespace_picto}{Picto} currently calls \hyperlink{class_picto_1_1_mouse_input_port_a67e15a3e712644d7a1bd70dcba94d818}{update\-Data\-Buffer()} once a frame, this hasn't been an issue. Since the polling timing is configured in the \hyperlink{class_picto_1_1_input_port}{Input\-Port} class, we simulate the correct number of samples when \hyperlink{class_picto_1_1_mouse_input_port_a67e15a3e712644d7a1bd70dcba94d818}{update\-Data\-Buffer()} is called even though it is actually only read once. To do this, we calculate the time since the last call to \hyperlink{class_picto_1_1_mouse_input_port_a67e15a3e712644d7a1bd70dcba94d818}{update\-Data\-Buffer()} and devide by the configured sample period for the number of samples needed, then copy the mouse data that number of times and enter it into the data buffer. \begin{DoxyAuthor}{Author}

Loads channel\-Buffers\-\_\- with data, returns the precise time when the data was actually read. 

In the case of the \hyperlink{class_picto_1_1_mouse_input_port}{Mouse\-Input\-Port}, there is no external hardware system or any external data buffer that is filled and needs to be copied in. Also, whenever the mouse is used as the input, precise timing is unnecessary. For this reason, we \char`\"{}simulate\char`\"{} reading in mouse data at the correct sample rate. We actually just check the mouse position once for every time that this function is called, but fake additional samples are produced to fill out the correct number of samples based on the configured sample rate and elapsed time since this function was last called.
