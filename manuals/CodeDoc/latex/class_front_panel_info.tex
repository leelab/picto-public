\subsection{Detailed Description}
Stores the command socket used by the Front Panel to communicate with the director as well as the current reward controller being used. 

This class used to be much more important. It could probably be removed and have its parts incorporated in other locations. In the meantime, it's still here. Note that the event\-Socket is no longer used. The \hyperlink{class_director_interface}{Director\-Interface} is the chief user of the command\-Socket and reward\-Controller data stored here. \begin{DoxySeeAlso}{See Also}
