\subsection{Detailed Description}
A state machine element for branching based on \hyperlink{class_picto_1_1_parameter}{Parameter} values. 

Flow Elements are used for branching in a state machine. Each flow element contains multiple \char`\"{}conditions\char`\"{} Each condition contains a predicate based expression (either a simple \hyperlink{class_picto_1_1_predicate_expression}{Predicate\-Expression}, or a mor complex \hyperlink{class_picto_1_1_compound_expression}{Compound\-Expression}) When the run is called on a flow element, it works throw the conditions (in a user defined order) until one of them evaluates true. The element then returns the name of the condition that evaluated true as its result. If none of the conditions evaluate true, then a default value is returned.
