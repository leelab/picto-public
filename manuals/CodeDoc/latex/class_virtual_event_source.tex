\subsection{Detailed Description}
An interface for a \hyperlink{class_virtual_event_source}{Virtual\-Event\-Source} that can be started and stopped, has a sampling\-Rate and can provide sample events in the form of \hyperlink{class_picto_1_1_data_unit}{Picto\-::\-Data\-Unit} objects. 

The key function here is get\-Next\-Event(time) which returns \hyperlink{class_picto_1_1_data_unit}{Picto\-::\-Data\-Unit} objects one by one, in time order until the input time arrives, at which point it should return a null pointer. \begin{DoxyAuthor}{Author}
