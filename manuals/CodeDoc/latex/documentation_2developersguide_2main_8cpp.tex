\subsection{Detailed Description}
This application was supposed to be used to navigate the \hyperlink{namespace_picto}{Picto} documentation. The application is really just a Qt brand web browser that looks at the regular \hyperlink{namespace_picto}{Picto} Html documentation. It is currently not working due to a simple configuration error and should be fixable by slightly adjusting the .pro file for the project. It practice though, this is a pretty pointless application. Just use a regular browser and look at output\textbackslash{}documentation\textbackslash{}main\textbackslash{}html\textbackslash{}index.\-html that way. \begin{DoxyAuthor}{Author}
