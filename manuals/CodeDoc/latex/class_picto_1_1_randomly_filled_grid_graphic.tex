\subsection{Detailed Description}
A graphic that contains a grid filled with two separate colors of squares. The positions of the different square colors is random and rerandomized at a preset frequency. The relative number of each color of squares is maintained over randomization. 

Randomly filled grids are used in many neuro physiology experiments. These are rectangular grids in which each space in the grid is filled with one of two colors, at random, according to a set distribution. The two colors, the number of squares in the grid and how it is split up into rows and columns can all be set. The designer can also set the rate at which the grid colors are re-\/randomized, and of course things like the grid size and position.

This is one of those graphics that was created toward the beginning of the \hyperlink{namespace_picto}{Picto} project but wasn't used in any actual experiments. For this reason it is missing some important things like Script Properties. It should really be brought up to date. \begin{DoxyAuthor}{Author}
Joey Schnurr, Mark Hammond, Matt Gay

Adds the following Properties\-:
\begin{DoxyItemize}
\item Color2\-: Every randomly filled grid graphic has two separate colors of squares, this is the second square color.
\item Size\-: The size of the grid graphic in pixels.
\item Horizontal\-Squares\-: The number of columns in the grid.
\item Vertical\-Squares\-: The number of rows in the grid.
\item Color2\-Squares\-: The number of squares that will have Color2 at a given time.
\item Frames\-Per\-Update\-: The number of frames that will pass before the grid is randomized and colored squares are redistributed. \begin{DoxyNote}{Note}
