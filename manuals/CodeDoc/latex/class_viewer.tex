\subsection{Detailed Description}
A \hyperlink{class_viewer}{Viewer} is a base class for panels in the Workstation that are used to handle some clearly defined job. 

Within Picto\-Workstation, you change modes by selecting different Viewers. For example, an Experiment Design can be viewed and edited in the \hyperlink{class_state_edit_viewer}{State\-Edit\-Viewer}. It can also be run in the \hyperlink{class_test_viewer}{Test\-Viewer}.

This base class implemements the minimum neccessary functionality\-: initialization, deinitialization, quitting and saving functionality are defined. Every viewer also has a pointer to a main Design\-Root since they all operate on the same main Design\-Root in some way.

When we change from one viewer to another, the process is\-:
\begin{DoxyEnumerate}
\item Call \hyperlink{class_viewer_ae36f82194449083e10825229a9e22fdf}{deinit()} on the old \hyperlink{class_viewer}{Viewer},
\item Wait for the old \hyperlink{class_viewer}{Viewer} to emit \hyperlink{class_viewer_aa8f10afad8094330d363080b275daaeb}{deinit\-Complete()}.
\item Call init on the new \hyperlink{class_viewer}{Viewer}
\item Change focus to the new \hyperlink{class_viewer}{Viewer}. \begin{DoxyAuthor}{Author}
