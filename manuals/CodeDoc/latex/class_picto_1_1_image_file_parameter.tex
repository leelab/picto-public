\subsection{Detailed Description}
An Image File parameter. 

This parameter holds an image file. Not the path to the file, but the actual file such that changing the original file on disk will not affect the contents of this parameter unless the file is reloaded. The class checks all loaded files to make sure that they have formats that \hyperlink{namespace_picto}{Picto} understands (it understands the formats that Q\-Image understands).

This class works with the \hyperlink{class_picto_1_1_image_graphic}{Image\-Graphic} class such that multiple \hyperlink{class_picto_1_1_image_graphic}{Image\-Graphic} objects can use the same \hyperlink{class_picto_1_1_image_file_parameter}{Image\-File\-Parameter}. \begin{DoxyAuthor}{Author}
