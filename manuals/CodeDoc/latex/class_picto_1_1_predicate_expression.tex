\subsection{Detailed Description}
A predicate expression involves a predicate and 2 values. 

A \hyperlink{class_picto_1_1_predicate_expression}{Predicate\-Expression} gets evaluated to a boolean value. It should be noted that the predicates and predicate expressions default to false. For example, if you tried to evaluate a predicate expression without having first set the left hand side value, it will automatically return false (as opposed to returning an error flag)
