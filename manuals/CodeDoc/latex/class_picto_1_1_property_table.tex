\subsection{Detailed Description}
A collection of all of an \hyperlink{class_picto_1_1_experiment}{Experiment}'s Properties used mainly for interfacing their values with the \hyperlink{namespace_picto}{Picto} \hyperlink{class_server}{Server}. 

The \hyperlink{class_picto_1_1_property_table}{Property\-Table} was once a big hash table containing all Properties indexed by their Asset\-Id. At a certain point we realized that the \hyperlink{class_picto_1_1_design_config}{Design\-Config} was the best place for handling that, and so the \hyperlink{class_picto_1_1_property_table}{Property\-Table} became essentially a fairly simple wrapper for a small portion of \hyperlink{class_picto_1_1_design_config}{Design\-Config} functionality. It might better have been called Property\-Update\-Interface or something like that since it essentially allows us to take a large group of unrelated Properties and simplify their interface so that all changed \hyperlink{class_picto_1_1_property}{Property} values can be condensed into a single list and a list of \hyperlink{class_picto_1_1_property}{Property} init\-Values that need to be set to their respective properties can be set by simply calling a single function in a loop.

This can be used in conjunction with the \hyperlink{namespace_picto}{Picto} Engine to send all current property values to server. \begin{DoxyNote}{Note}
This function is set to ignore \hyperlink{class_picto_1_1_u_i_info}{U\-I\-Info} objects, Transitions and Control Links. Transitions and Control Links don't have any Properties that can change during and experiment. \hyperlink{class_picto_1_1_u_i_info}{U\-I\-Info} objects have no bearing on the experiment and are used only for U\-I information for the \hyperlink{class_picto_1_1_state}{State} Machine Editor. 

Whereas in \hyperlink{class_picto_1_1_property_table_aa48dc61b2d97049641d13a66c556168a}{add\-Property()} we wrote that only those Properties added through that function would participate in value\-Changed() signal emission. That was with regard to when the \hyperlink{class_picto_1_1_property}{Property} values actually change. In this function, we emit the value\-Changed() signals regardless of what was or was not added in \hyperlink{class_picto_1_1_property_table_aa48dc61b2d97049641d13a66c556168a}{add\-Property()}.
