\subsection{Detailed Description}
A simple list for holding error messages and their types. 

This is a very simple object. Errors are added to the list with \hyperlink{class_error_list_a0d7142914d8cf6d4770af60ad9a5855f}{add\-Error()}, they are accessed with \hyperlink{class_error_list_aa6bf2297c742d616cd0f45018d0b41b4}{get\-All\-Errors()} and the list is cleared with \hyperlink{class_error_list_a525741e6e6ae8c20af823e6b9ce8185c}{clear()}. \begin{DoxyAuthor}{Author}
