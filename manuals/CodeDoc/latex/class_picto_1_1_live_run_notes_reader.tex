\subsection{Detailed Description}
Implements the \hyperlink{class_picto_1_1_run_notes_reader}{Run\-Notes\-Reader} class for a live test experiment. 

Since run notes are pretty meaningless in a test experiment, but we want \hyperlink{class_picto_1_1_analysis}{Analysis} elements that poll run notes data to return something, this class pretty much just extends the \hyperlink{class_picto_1_1_run_notes_reader}{Run\-Notes\-Reader} functions to return simple fake data like the \char`\"{}\-Sample Name\char`\"{} name and the \char`\"{}\-Sample Notes\char`\"{} notes. Start time is zero and end time is always -\/1 (since we don't know how long the run is while we're still in it). Run index is zero. \begin{DoxyAuthor}{Author}
