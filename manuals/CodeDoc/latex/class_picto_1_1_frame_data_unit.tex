\subsection{Detailed Description}
Stores information about rendering of a single frame. 

Stored data is the first phosphor time and the Asset\-Id of the \hyperlink{class_picto_1_1_state}{State} that was active when the frame was rendered. \begin{DoxyNote}{Note}
In this and some other Data\-Units, we store a timestamp as a string to be sure that no one reduces the precision of the Timestamp anywhere in transit (as might happen if some other class converted a timestamp double to a string with unknown precision). 
\end{DoxyNote}
