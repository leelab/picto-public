\subsection{Detailed Description}
A box that acts as a port for the tail or head of a transition arrow. 

This \hyperlink{class_arrow_port_item}{Arrow\-Port\-Item} takes care of things like telling the arrow connected to it if it needs to redraw itself with new endpoints and deleting an attached arrow if this \hyperlink{class_arrow_port_item}{Arrow\-Port\-Item} is deleted. \begin{DoxyAuthor}{Author}
Joey Schnurr, Mark Hammond, Matt Gay
