\subsection{Detailed Description}
This is a base class for all classes that can be used to access detailed data about a \hyperlink{namespace_picto}{Picto} Session. 

When analyzing a \hyperlink{namespace_picto}{Picto} session, beyond simply retrieving the current state of the \hyperlink{class_picto_1_1_state_machine}{State\-Machine}, we need the capability to read the values of input signals and neural data. That is the purpose of \hyperlink{class_picto_1_1_analysis_data_source}{Analysis\-Data\-Source} objects, these object are used to retrieve additional data about the experimental run beyond what is immediately readable from the state of the regular experimental variables.

Currenlty, this class does not add any functionality to \hyperlink{class_picto_1_1_analysis_variable}{Analysis\-Variable}, but we wanted to create this base class since its children are conceptually similar and it may be useful to add general functionality for them in the future. \begin{DoxyAuthor}{Author}
