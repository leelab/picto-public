\subsection{Detailed Description}
A \hyperlink{class_picto_1_1_control_target_result}{Control\-Target\-Result} is used by \hyperlink{class_picto_1_1_choice_controller}{Choice\-Controller}. 

A \hyperlink{class_picto_1_1_control_target_result}{Control\-Target\-Result} is used by \hyperlink{class_picto_1_1_choice_controller}{Choice\-Controller} for defining the \hyperlink{class_picto_1_1_control_target}{Control\-Target} elements that the subject may choose from.

Each \hyperlink{class_picto_1_1_control_target_result}{Control\-Target\-Result} allows the designer to select a single \hyperlink{class_picto_1_1_control_target}{Control\-Target}. If that \hyperlink{class_picto_1_1_control_target}{Control\-Target} is selected, that \hyperlink{class_picto_1_1_control_target_result}{Control\-Target\-Result} will be set as the \hyperlink{class_picto_1_1_choice_controller}{Choice\-Controller} element's result. \begin{DoxyAuthor}{Author}
