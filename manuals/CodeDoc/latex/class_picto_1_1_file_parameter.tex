\subsection{Detailed Description}
A base class for File Parameters that store File contents. 

This parameter holds a file. It also holds the path to the file. Don't be fooled though, this is storing the actual file contents such that changing the original file on disk will not affect the contents of this parameter unless the file path is reset.

Whenever the File\-Name \hyperlink{class_picto_1_1_property}{Property} changes, this class automatically reads the contents of the file at the File\-Name path and stores them in a special X\-M\-L friendly Base64 format in the Design file. This way no matter where the Design.\-xml goes, the File will come with it, there will be no need to make sure that file resources are at proper paths to get the design to run. \begin{DoxyAuthor}{Author}
Joey Schnurr, Mark Hammond, Matt Gay
