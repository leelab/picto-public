\subsection{Detailed Description}
This is the base class for all graphical elements in the \hyperlink{namespace_picto}{Picto} \hyperlink{class_designer}{Designer} apart from transition arrows. 

Objects of this class draw a rectangle on the \hyperlink{class_designer}{Designer} canvas. They provide an option to set a name and type for the rectangle that are drawn onto it. The dimensions of the rectangle can be set and interfacing with the \hyperlink{class_editor_state}{Editor\-State} object that manages the current state of the \hyperlink{class_designer}{Designer} is handled automatically (Double clicking a \hyperlink{class_diagram_item}{Diagram\-Item}, for example, sets it as the window asset). The \hyperlink{class_diagram_item}{Diagram\-Item} also supports colored oulines and highlighting of the \hyperlink{class_diagram_item}{Diagram\-Item} name (which the \hyperlink{class_designer}{Designer} uses in searches). Diagram\-Items also support 'right-\/click' context menus.
