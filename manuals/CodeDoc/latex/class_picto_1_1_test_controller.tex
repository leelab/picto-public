\subsection{Detailed Description}
A simple controller for debugging purposes. This controller is not available in the design viewer. 

This controller is a wrapper that can be modified (in C++) to do whatever is needed for test purposes Currently, it is simply outputing to the debuggin console, and return \hyperlink{class_picto_1_1_test_controller_a60da5e71238caac387ff2c43dbc2e6df}{is\-Done()} == true the 120th time it is called (2 seconds if we're running at 60\-Hz). \begin{DoxyAuthor}{Author}
Joey Schnurr, Mark Hammond, Matt Gay
