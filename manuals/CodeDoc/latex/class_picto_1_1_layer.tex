\subsection{Detailed Description}
A \hyperlink{class_picto_1_1_layer}{Layer} contains a set of Visual\-Elements. 

Layers are rendered in order from smallest to largest. Each \hyperlink{class_picto_1_1_layer}{Layer} contains multiple Visual\-Elements which make up the layer.
