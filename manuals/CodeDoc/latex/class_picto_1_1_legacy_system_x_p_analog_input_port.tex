\subsection{Detailed Description}
An \hyperlink{class_picto_1_1_input_port}{Input\-Port} object for collecting analog input data when running \hyperlink{namespace_picto}{Picto} on a Legacy\-System\-X\-P (ie. an old Orion machine). 

This \hyperlink{class_picto_1_1_input_port}{Input\-Port} is designed to collect analog data from the inputs to the N\-I\-D\-A\-Q board in the Legacy\-System.

The following assumptions are made in the code\-:
\begin{DoxyItemize}
\item Legacy\-System is running Windows X\-P
\item Legacy\-System has a P\-C\-I-\/\-M\-I\-O-\/16\-X\-E-\/50 card installed, with some number of A\-I channels connected
\item The D\-A\-Q card is \char`\"{}\-Dev2\char`\"{}
\item The development box has the N\-I\-D\-A\-Qmx libraries and headers installed \begin{DoxyAuthor}{Author}
