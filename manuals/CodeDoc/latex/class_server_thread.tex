\subsection{Detailed Description}
A thread used to handle T\-C\-P connections. 

When a T\-Cp connection is created by the server, a new \hyperlink{class_server_thread}{Server\-Thread} is spun up to handle the incoming data. The thread reads the incoming data as if it is following H\-T\-T\-P protocol. It then finds the appropriate command handler (using the protocol from \hyperlink{struct_server_protocols}{Server\-Protocols} and the command name) and calls that command handler for a response which is then returned to the sender. \begin{DoxyAuthor}{Author}
