\subsection{Detailed Description}
A parameter for storing strings that is not being used. 

There is a \hyperlink{class_picto_1_1_string_variable}{String\-Variable} for this purpose, so having a separate \hyperlink{class_picto_1_1_parameter}{Parameter} to do it doesn't seem to useful although one could imagine things like showing text messages to test subjects that are updatable during an experiment from the Remote \hyperlink{class_viewer}{Viewer}. This should be easy to implment someday if someone feels the need to do so. \begin{DoxyAuthor}{Author}
