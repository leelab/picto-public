\subsection{Detailed Description}
A base class for storing values that can be used in a state machine. 

Since \hyperlink{namespace_picto}{Picto} requires users to build state machines to define their experiments, we need some sort of \char`\"{}variable\char`\"{} to hold data over defined scopes. Descendants of this class define those variables. \begin{DoxyAuthor}{Author}
