\subsection{Detailed Description}
A Simple Event Source that builds Neural\-Data\-Unit objects. 

The frequency at which the Neural\-Data\-Unit objects will be created along with the range over which their timespans are randomized is set as the sec\-Per\-Event value in the constructor. channel and unit for this source are also set there. Apart from that, objects of this class are used entirely for their \hyperlink{class_simple_spike_source_a42730994330acdf1d66e3b232d86f276}{build\-Event()} function which just creates Neural\-Data\-Unit objects one after the other when \hyperlink{class_simple_event_source}{Simple\-Event\-Source} says to do so. \begin{DoxyAuthor}{Author}
Joey Schnurr, Mark Hammond, Matt Gay 
\end{DoxyAuthor}
