\subsection{Detailed Description}
\hyperlink{class_picto_1_1_analysis_output}{Analysis\-Output} objects are the endpoint of \hyperlink{class_picto_1_1_analysis}{Analysis} data. Any data captured and manipulated in the \hyperlink{class_picto_1_1_analysis}{Analysis}, will finally be written to an \hyperlink{class_picto_1_1_analysis_output}{Analysis\-Output} for presentation to the viewer. 

An \hyperlink{class_picto_1_1_analysis_output}{Analysis\-Output} might be as simple as an \hyperlink{class_picto_1_1_analysis_file_output}{Analysis\-File\-Output} with which text can be written to a file. It might also be as complex as a specific type of raster output which might output a pdf file containing a raster plot. Data is written to \hyperlink{class_picto_1_1_analysis_output}{Analysis\-Output} objects with simple javascript functions exposed by the each type of \hyperlink{class_picto_1_1_analysis_output}{Analysis\-Output} class just like any other element in a \hyperlink{namespace_picto}{Picto} design.

\hyperlink{class_picto_1_1_analysis_output}{Analysis\-Output} objects should also be associated with an \hyperlink{class_picto_1_1_analysis_output_widget}{Analysis\-Output\-Widget} that is used to present their data to the operator during an \hyperlink{class_picto_1_1_analysis}{Analysis}. The specific type of \hyperlink{class_picto_1_1_analysis_output}{Analysis\-Output} class is responsible for defining the correct type of widget to be used with it. \hyperlink{class_picto_1_1_analysis_output}{Analysis\-Output} objects are responsible for creating their associated widget and filling it with data. The \hyperlink{class_picto_1_1_analysis_output}{Analysis\-Output} class keeps track of all of the \hyperlink{class_picto_1_1_analysis_output}{Analysis\-Output} objects that are participating in a particular run. This allows it to create an \hyperlink{class_picto_1_1_analysis_output_widget_container}{Analysis\-Output\-Widget\-Container} that packages all of the Analysis\-Output\-Widgets into a single parent widget for display in the \hyperlink{namespace_picto}{Picto} environment.

Data saved by \hyperlink{class_picto_1_1_analysis_output}{Analysis\-Output} objects is stored underneath the \char`\"{}analysisoutput\char`\"{} directory in the \hyperlink{namespace_picto}{Picto} run directory. Each time \hyperlink{namespace_picto}{Picto} starts, this directory is erased meaning that storage there is temporary. \hyperlink{class_picto_1_1_analysis_output_widget}{Analysis\-Output\-Widget} objects take care of copying data from that temporary directory to a permanent location on the operator's request. \begin{DoxyAuthor}{Author}
