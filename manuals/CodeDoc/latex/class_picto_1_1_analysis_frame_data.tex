\subsection{Detailed Description}
An \hyperlink{class_picto_1_1_analysis_data_source}{Analysis\-Data\-Source} for gathering frame timing data. 

This object can be used to query the latest, next and previous frame times as well as lists of frame times for input time ranges.

All frame times are the \hyperlink{namespace_picto}{Picto} system's best estimate for the time at which the first phosphor of the frame appeared on screen. When the frame is displayed with Direct\-X as in the \hyperlink{class_director}{Director}, this estimate is fairly precise (within a few ms) since Direct\-X can be polled for that type of information. When using Pixmap rendering, the value is less precise. \begin{DoxyNote}{Note}
This class uses an \hyperlink{class_picto_1_1_lfp_reader}{Lfp\-Reader} as its main data source. 
\end{DoxyNote}
