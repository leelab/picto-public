\subsection{Detailed Description}
A Simple Event Source that builds Alignment\-Data\-Unit objects. 

The frequency at which the Alignment\-Data\-Unit objects will be created is set in the constructor. Apart from that, objects of this class are used entirely for their \hyperlink{class_simple_mark_source_a233577f961e2911124ae959809698b2a}{build\-Event()} function which just creates Alignment\-Data\-Unit objects one after the other with increasing Alignment codes that wrap to 1 at 128. \begin{DoxyAuthor}{Author}
Joey Schnurr, Mark Hammond, Matt Gay 
\end{DoxyAuthor}
