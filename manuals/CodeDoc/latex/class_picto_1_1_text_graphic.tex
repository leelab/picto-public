\subsection{Detailed Description}
Displays text in the \hyperlink{class_picto_1_1_experiment}{Experiment} \hyperlink{class_picto_1_1_scene}{Scene}. 

Obviously this wouldn't be used with \char`\"{}\-Subject\-View\char`\"{} enabled for Experiments with non-\/human subjects, but it is very useful with \char`\"{}\-Operator\-View\char`\"{} for debugging and providing info on the operator screen.

At the moment, a \hyperlink{class_picto_1_1_text_graphic}{Text\-Graphic} is limited to 12 point Times, but there is no reason we couldn't expand its capablilites at some point.

Beyond the Properties inherited from \hyperlink{struct_picto_1_1_visual_element}{Visual\-Element}, this class adds a Size \hyperlink{class_picto_1_1_property}{Property} that is used for defining the Size of the Window in which text appears. It is important to remember that if the Size is too small, text isn't going to shrink to fit it, it will just run off the end and be invisible. For this reason, it is important to always be sure that the \hyperlink{class_picto_1_1_text_graphic}{Text\-Graphic} has a large enough Size to handle any text that we expect to be written. \begin{DoxyAuthor}{Author}
Joey Schnurr, Mark Hammond, Matt Gay
