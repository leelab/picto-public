\subsection{Detailed Description}
This is a top level container acting as a single location to access all of the various objects that are part of the rendering system for a specific type of rendering hardware. 

\hyperlink{struct_picto_1_1_rendering_target}{Rendering\-Target} acts as a virtual base class for specialization to specific types of rendering hardware. This could be pure software (e.\-g. for the web interface, movie file generation, printing, etcetera), pixmap based (e.\-g. for the workstation), D3\-D based (e.\-g. for the director on a full Windows machine), Direct\-Draw based (e.\-g. for the director on a Picto\-Box), or null based (e.\-g. for analysis).

Each \hyperlink{struct_picto_1_1_rendering_target}{Rendering\-Target} contains a \hyperlink{class_picto_1_1_visual_target}{Visual\-Target} object and an \hyperlink{class_picto_1_1_aural_target}{Aural\-Target} object both of which act as factory classes for elements that can then be combined by the renderer to produce the final output. Convenience functions are also available for creating \hyperlink{struct_picto_1_1_compositing_surface}{Compositing\-Surface} and \hyperlink{class_picto_1_1_mixing_sample}{Mixing\-Sample} objects directly rather than going through these \hyperlink{class_picto_1_1_visual_target}{Visual\-Target} and \hyperlink{class_picto_1_1_aural_target}{Aural\-Target} classes.
