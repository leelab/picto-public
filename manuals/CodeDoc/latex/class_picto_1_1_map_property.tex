\subsection{Detailed Description}
A \hyperlink{class_picto_1_1_property}{Property} for storing Map (lookup table) data. This is used for storing a map of key, value pairs. A number of \hyperlink{class_picto_1_1_data_store}{Data\-Store} objects need to store key value pair structures. A Variant\-Map. for example, needs to do this by its very definition. This is the \hyperlink{class_picto_1_1_property}{Property} that is used for that purpose. 

\begin{DoxyNote}{Note}
Typically, it doesn't make sense to make a \hyperlink{class_picto_1_1_map_property}{Map\-Property} editable by the designer or \hyperlink{class_picto_1_1_experiment}{Experiment} operator. \hyperlink{class_picto_1_1_map_property}{Map\-Property} are usually set invisible to the G\-U\-I (\hyperlink{class_picto_1_1_property_ad55eb788c4d635b92fa5944c78a1d346}{set\-Visible()}) and used exclusively as a way to describe an \hyperlink{class_picto_1_1_data_store}{Data\-Store}'s current state to support the requirement that that state be fully described by the \hyperlink{class_picto_1_1_data_store}{Data\-Store}'s Properties. 
\end{DoxyNote}
