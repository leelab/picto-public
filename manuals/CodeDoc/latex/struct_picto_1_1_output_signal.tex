\subsection{Detailed Description}
A Base class for Output\-Elements that output voltage signals. 

Frequently, an experiment requires some kind of means to interface with outside devices using T\-T\-L signals. This is the purpose of the \hyperlink{struct_picto_1_1_output_signal}{Output\-Signal} class. Descendants of \hyperlink{struct_picto_1_1_output_signal}{Output\-Signal} allow the Experimental Design to control the physical voltage of Ports on the computer running the \hyperlink{class_director}{Director}. In this base class, we implement functionality for setting the physical port handled by the \hyperlink{struct_picto_1_1_output_signal}{Output\-Signal} as well as enabling/disabling the port. Enabling/disabling a port sets whether the value set in this \hyperlink{struct_picto_1_1_output_signal}{Output\-Signal} will actually physically appear on the port or not. It is analogous to visible/invisible in a \hyperlink{struct_picto_1_1_visual_element}{Visual\-Element}. \begin{DoxyAuthor}{Author}
