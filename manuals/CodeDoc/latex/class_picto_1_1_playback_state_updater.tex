\subsection{Detailed Description}
Extends \hyperlink{class_picto_1_1_state_updater}{State\-Updater} to update the \hyperlink{namespace_picto}{Picto} \hyperlink{class_picto_1_1_state}{State} based on a Session file. 

The Session file path is set in \hyperlink{class_picto_1_1_playback_state_updater_adc11cc6489dd7d46067c2e58bebb167a}{set\-File()}. A \hyperlink{class_picto_1_1_file_session_loader}{File\-Session\-Loader} is used with a \hyperlink{class_picto_1_1_session_state}{Session\-State} to actually load the file and a \hyperlink{class_picto_1_1_session_player}{Session\-Player} is used to playback the \hyperlink{class_picto_1_1_session_state}{Session\-State}.

This class implements things like pausing, stopping, playing, etc and defines the correct times to enter into the \hyperlink{class_picto_1_1_session_player}{Session\-Player} accordingly. It implements \hyperlink{class_picto_1_1_state_updater}{State\-Updater} so that it can be used with a \hyperlink{class_picto_1_1_slave_experiment_driver}{Slave\-Experiment\-Driver} to drive a slave's experimental state. \begin{DoxyAuthor}{Author}
Joey Schnurr, Mark Hammond, Matt Gay

2009-\/2013
