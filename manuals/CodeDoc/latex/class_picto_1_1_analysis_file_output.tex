\subsection{Detailed Description}
A simple \hyperlink{class_picto_1_1_analysis_output}{Analysis\-Output} object used for writing text or binary data to files. 

This object will take any text or binary data written through its \hyperlink{class_picto_1_1_analysis_file_output_afd357d2fb660be88ad81d589720bd123}{write\-Line()}, \hyperlink{class_picto_1_1_analysis_file_output_a3a9ea552a091035c48e135f248d6433c}{write\-Text()} or \hyperlink{class_picto_1_1_analysis_file_output_aa4249731a0bfdca39fe880af06717f07}{write\-Binary()} functions and write it to a file. The filename is the same as the current run name, and the file suffix is settable as a \hyperlink{class_picto_1_1_property}{Property}. The file can be set as a text file or a binary file, with both big-\/endian and little-\/endian binary options.

For in \hyperlink{namespace_picto}{Picto} viewing of output data, this object returns an Analysis\-File\-Widget or Analysis\-Binary\-Widget from create\-Widget depending on the file type. \begin{DoxySeeAlso}{See Also}
