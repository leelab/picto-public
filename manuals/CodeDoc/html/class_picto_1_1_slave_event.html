Detailed Description

Holds information about an experimental state change event. Used by SlaveExperimentDriver.

Property value, Property init value and Transition activation event times are included. Each SlaveEvent has a type, an ID representing the affected Property or Transition's Asset ID, and a string value which is not used by TRANS_ACTIVATED events but holds a Property's value/initValue for PROP_VAL_CHANGE and INIT_PROP_VAL_CHANGE events.
