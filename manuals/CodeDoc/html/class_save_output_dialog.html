Detailed Description

A dialog for saving analysis output.

Qt provides some really easy ways to do a save file dialog, but we couldn't use them for saving output because we wanted to add an option for saving Output files into separate directories by run name instead of putting them all into the same directory. For that reason, we made this custom SaveOutputDialog that includes that extra checkbox.
