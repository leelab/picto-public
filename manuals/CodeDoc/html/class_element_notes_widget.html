Detailed Description

A widget used to enter notes that are saved with elements of the design. This is how comments are entered in Picto Designs.

Obviously it is extremely important to comment code. This is how we "comment code" in the Picto Design. Text entered into this widget is saved with the currently selected element and displayed in its tooltip. It is also searched when a search string is typed into the Picto search box.
