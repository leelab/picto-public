34  //for additional detail.)
35  //The WAL journaling mode uses a write-ahead log instead of a rollback journal to implement

37  //multiple database connections and after closing and reopening the database. A database in WAL
38  //journaling mode can only be accessed by SQLite version 3.7.0 or later. This mode can be significantly
39  //faster than DELETE in cases where there are significantly more writes than reads. This mode uses
