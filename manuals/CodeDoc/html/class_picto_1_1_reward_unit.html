Detailed Description

Stores information about the default reward delivery time on a particular channel when a simple reward button press occurs.

This stores the default duration of a reward on the stored channel. This is the quantity supplied when the reward button is pressed on the Pictobox or the reward button on the workstation is pressed. These systems are synchronized by using this DataType such that they always include the same reward quantity.
