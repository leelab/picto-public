Detailed Description

An ArrowSourceItem representing a result that hangs off of a diagram representing the StateMachineElement which contains it.

DiagramItems representing StateMachineElements show all of their child results hanging off of the bottom. This item represents one of these results. It is an ArrowSourceItem and contains a ScriptItemManager to display the EntryScript of the Result that it represents.
