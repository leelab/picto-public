Detailed Description

Stores a list of StateDataUnit objects representing StateMachine Transitions between one frame and the next.

Objects of this class store the entire StateMachine traversal that occured during the course of a single frame. The traversal is stored as a list of StateDataUnit objects each of which contains a Transition AssetId. These StateDataUnit objects are stored with the DataId of the Frame that followed them, and since they each have their own DataId, we can follow the order of Transition traversals accordingly.
