Detailed Description

A status light widget.

We couldn't find any good stock status light widgets so we made our own. The status light is used in the proxy server's GUI. It is just a circle that changes colors (red, yellow, green) to indicate status. This was an early attempt at writing a Qt GUI, so it's a bit crude.
