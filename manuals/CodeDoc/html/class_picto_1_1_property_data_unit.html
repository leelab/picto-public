Detailed Description

Stores data about a particular change to a Property's value.

This class holds data about a single Property value change. The Property whose value changed is referenced by its AssetId. The changed value can be either a runValue or an initValue, as indicated by the initValue_ field. The Value itself is stored in a string form that can be converted to a Property value by using Property::valFromUserString().
