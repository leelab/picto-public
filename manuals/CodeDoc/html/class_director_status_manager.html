Detailed Description

Handles overall status operations for a Director object.

This class adds to the default ComponentStatusManager functionality by implimenting Status and user info display for the Experiment subject, interfacing with any Control Panels attached to the Director, sending Alignment Events to attached neural systems, and sending COMPONENTUPDATE commands and Alignment Event data to the Picto Server.
