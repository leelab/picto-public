Detailed Description

A compositing surface based on an underlying QPixmap data structure.

QPixmap is a basic bitmap impage. It can be painted to, and is really easy to work with. The downside is that it is really slow (not surprising for a software bitmap). The PixmapCompositingSurface would probably not be used in PictoDirector, but it is multiplatform compatible and therefore used in the PictoWorkstation so that the user can see what is being displayed by a Director instance.
