Detailed Description

The base class for all logic predicates (examples: ==, <. >, etc)

In a state machine, we are constantly evaluating logical expressions. For example the expression "trial < 10" might be used to determine a branching path Since the state machine is GUI based, and does not involve writing code, we are using predicates, predicateExpressions, and CompoundExpressions to control these branches.

The predicate class is a base class which all predicates must inherit.

Possible predicates: < > <=

>=

!= javascript
