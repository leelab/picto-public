Detailed Description

A syntax highlighter for Picto Scripts.

Since there is no Qt API ScriptSyntaxHighlighter, we used a file from the QtScriptTools source for our own local QScript highlighting this code comes almost entirely from their. Presumably, in the future a standard ScriptSyntaxHighlighter will be available in the API at which point we can migrate to that one.

We will not include detailed documentation of this class, since we basically never looked at its code, pretty much all we do is call its constructor with the document that we want to highlight and it takes care of the rest.
