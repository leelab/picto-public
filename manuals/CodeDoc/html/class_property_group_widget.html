Detailed Description

This Widget creates groups of Properties widgets. It groups together widgets for setting/getting values of a Properties and labels each group with a title.

Properties for display are passed in using the addProperties() function along with a title for each Property group.

This widget's job is essentially just creating PropertyListWidgets which take care of actually creating property widgets and displaying them and then placing each of those widget within a group layout.
