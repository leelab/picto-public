Detailed Description

A base class that represents an area with a drawing on it.

QImages are converted and drawn onto CompositingSurface objects. Multiple CompositingSurface objects are in turn drawn onto a VisualTarget which is what is finally rendered on screen. In essence, a CompositingSurface can be thought of as an image object that is drawn in a language that the VisualTarget understands, or looked at another way, a compositing surface is a translater, translating from the language of the QImage to the language of the VisualTarget.

void Picto::CompositingSurface::dontAllowScaling ( )
inline

Sets this CompositingSurface as non-scalable.

This is useful for graphics that are overlayed on top of the display such as the user position cursor. When zooming into the current display, we want all of the displayed graphics to be zoomed, but the cursor and other overlayed graphics should stay the same size.
