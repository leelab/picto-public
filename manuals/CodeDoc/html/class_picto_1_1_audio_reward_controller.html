Detailed Description

A RewardController that simulates rewards using an audio signal for testing purposes.

In the Test Viewer, we don't have a real rewarder connected. Instead, we use this rewarder to generate an audible stimulus every time a reward is given. The audio stimulus is a tick noise that is included in the Picto binaries. It is currenly in the Workstation source under a sounds directory. It should really be moved into the common directory.
