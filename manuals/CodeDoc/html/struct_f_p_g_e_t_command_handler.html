Detailed Description

A Picto::ProtocolCommandHandler that Handles FPGET commands received by the Director from the EmbeddedFrontPanel.

Returns the requested data (IPAddress, DirectorName, DirectorStatus, RewardDuration, FlushDuration).

Rather than creating some kind of complex circular reference system, we have simply exposed functions here that higher level classes can use to update this ProtocolCommandHandler with the data that it needs.
