Detailed Description

Stores Information about a paricular data source (ie. lfp or spike)

Stored data is the name of the data source, the name of the table to which its data should be saved in the Session file, and the resolution (really this is the sample period) of sampling on that data source.

Note
In the case of spikes, resolution is the time per sample for a spike's waveform.
