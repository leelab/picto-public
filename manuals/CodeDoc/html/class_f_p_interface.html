Detailed Description

The interface between the Picto Director and the application managing the Pictobox Front Panel.

This class implements the ControlPanelInterface for communication with the EmbeddedFrontPanel application. The interface is through a local socket and simple http style commands (ie. FPPUT, FPGET, etc).
