Detailed Description

A drop down menu for selecting Analyses for the Designer.

This widget is used to control Analysis editing in the Designer. Originally it was also responsible for selecting Analysis when running an experiment, but more flexibility was required for that, so the AnalysisSelectorWidget was developed.
