Detailed Description

This adds video file recording functionality onto the regular VisualTargetHost functionality.

This class extends the VisualTargetHost so that whenever a new frame is drawn to its VisualTarget it is also writtent to a video file. Functions are provided for enabling/disabling the "copy frame to video file" functionality. saveRecordingAs() can be used to save the recorded video file to a particular path on disk.

Note
This class currently uses qtffmpegwrapper which is a wrapper for ffmpeg since there is no way yet to save video files directly from Qt. qtffmpegwrapper will undoubtedly need to be updated at some point. It can be found in Picto\3rdParty.
