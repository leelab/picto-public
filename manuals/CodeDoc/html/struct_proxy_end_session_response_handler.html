Member Function Documentation

bool ProxyEndSessionResponseHandler::processResponse ( QString  directive)
protectedvirtual

Implements ProtocolCommandHandler::processCommand() to take care of ending the current session.

This function sets the ComponentStatusManager status to "ending", then just calls CommandChannel::processResponses() over and over until that function finishes sending all the data that it needs to send to the Server with verification that all of the data was recieved and saved. Once that is all done, the function returns and the Session is over.
