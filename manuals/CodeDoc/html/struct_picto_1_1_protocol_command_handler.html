Detailed Description

A base class from which all command handlers are derived. These command handlers define responses to ProtocolCommand objects coming in over the network.

A command handler needs to implement two functions. The method function is simply the name of the command (e.g. GET). The processCommand function is where all of the logic is. The processCommand function should return a response, and cause any necessary internal state changes.
