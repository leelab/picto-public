Detailed Description

A base class for ProtocolResponseHandler classes that respond to the "STOP" method.

A STOP response indicates that the currently running Task should be stopped.

The processResponse() function implemented by this class is empty. Extend this class to add logic that is triggered by the arrival of a new ProtocolRespose.
