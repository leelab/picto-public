Detailed Description

A DisplayMode that allows the user to monitor the current status of the Front Panel application.

Displayed information includes Pictobox Name, ServerDisconnected/ServerConnected/Stopped/Running status symbol, Reward duration, Flush duration, Default Reward Controller Number, PictoBox IP address. This data takes up three lines which the user navigates by turning the input knob.
