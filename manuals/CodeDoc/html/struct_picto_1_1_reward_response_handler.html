Detailed Description

A base class for ProtocolResponseHandler classes that respond to the "REWARD" method.

A REWARD response indicates that a reward should be supplied to the Experiment subject. The message includes the reward channel on which the reward should be supplied and its duration.

The processResponse() function implemented by this class is empty. Extend this class to add logic that is triggered by the arrival of a new ProtocolRespose.
