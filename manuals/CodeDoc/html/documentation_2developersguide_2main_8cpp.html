Detailed Description

This application was supposed to be used to navigate the Picto documentation.

The application is really just a Qt brand web browser that looks at the regular Picto Html documentation. It is currently not working due to a simple configuration error and should be fixable by slightly adjusting the .pro file for the project. It practice though, this is a pretty pointless application. Just use a regular browser and look at output\documentation\main\html\index.html that way.
