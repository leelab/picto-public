Detailed Description

A Syntax Error widget used to display the output of the Picto Design Validation (ie. compilation) process.

The Picto design is checked using DesignRoot::compiles(). If the design syntax is invalid, this can result in error messages. These error messages are displayed in this widget.

See Also
DesignRoot::compiles(), Asset::validateTree()
