Detailed Description

A base class for ProtocolResponseHandler classes that respond to the "RESUME" method.

A RESUME response indicates that a paused Experiment should resume operation.

The processResponse() function implemented by this class is empty. Extend this class to add logic that is triggered by the arrival of a new ProtocolRespose.
