Detailed Description

An AssetItem used to represent Experimental (ie. Non-Analysis) elements.

This is pretty much just an AssetItem that is colored greyish to represent that it is part of the experiment and not the analysis. It also has functionality for disabling certain capabilities when the Designer is in "Analysis design" mode.

See also
EditorState::setCurrentAnalysis()
