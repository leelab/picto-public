Detailed Description

A DiagramItem used to display DataStores.... it should probably have been called DataStoreItem.

Represents a Diagram DataStore with a name, type and icon image.

Author
Joey Schnurr, Mark Hammond, Matt Gay

Member Function Documentation

void AssetItem::assetEdited ( )
slot

Called when the underlying Asset is edited to update the displayed information accordingly.

The AssetItem shows the Asset's name, type, notes, and highlights the search string in its name. Most of these aspects can be changed during design development, so this needs to take care of updating the UI accordingly.
