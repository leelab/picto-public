Detailed Description

Handles overall status operations for a ProxyMainWindow object.

This class adds to the default ComponentStatusManager functionality by implimenting doServerUpdate() to send COMPONENTUPDATE commands to the Picto Server. It also provides functions for setting/getting the neural data acquisition plugin that will be used to gather data from the Neural DAQ software for sending to the Server.
