Detailed Description

A DataUnit used to send serialize/deserialize alignment data.

Alignment events are 7 bit codes sent from the Director to the Proxy to synchronize their timing. Both systems mark the time at which the alignment code occured, and the Directer, which generates the code also records the index of the event (Alignment Number, not limited to 7 bits). They can then be matched up and used for a least squares method of time synchronization.
