Detailed Description

Handles reading and processing incoming commands NO LONGER USED.

This object used to allow the director to push data to the FrontPanel application. It is no longer used. The front panel is now set up as the communications master handling both setting and polling data. At a more opportune time, this object should probably be deleted and references to it removed from other parts of the application.
