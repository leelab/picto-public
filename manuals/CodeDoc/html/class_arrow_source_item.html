Detailed Description

An ArrowPortItem to which Arrow tails attach.

This is a base class for result bars that hang of the bottom of each StateMachineElement diagram as well as the StartBar on the left side of the canvas.

Author
Trevor Stavropoulos, Joey Schnurr, Mark Hammond, Matt Gay
Date
2009-2015
