Detailed Description

The base class of widgets that display a representation of the output of an AnalysisOutput object.

These widgets are displayed in an AnalysisOutputWidgetContainer for monitoring of Analysis outputs from within the Picto application. Different AnalysisOutput classes will be associated with different AnalysisWidgets, for example, an AnalysisFileOutput will be associated with an AnalysisFileOutputWidget, whereas an AnalysisRasterOutput might still save its data to file, but be viewable using an AnalysisRasterPlotWidget.
