Detailed Description

A simple button widget containing an up arrow that turns red if the current Window Asset's ancestors contain the current search query.

This widget is used by the Designer to switch the Window Asset to the parent of the current Window Asset.

See also
EditorState::getWindowAsset(), EditorState::setWindowAsset(), EditorState::setWindowAssetToParent()
